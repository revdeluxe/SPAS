# SPAS
- Smart Public Address System is an application that makes use of JavaScript using NodeJS as its base.
___With the use of browsers the ease of deploy SPAS is easy as CTRL + C & CTRL + V____
# Dependencies
1. NodeJS
2. JsSIP
3. requireJS
4. httpsJS
